# Luma Bot
Luma Bot - телеграм-бот для онлайн-магазина Luma Store. Он интуитивно понятен в управлении для пользавателей, а также для администраторов.
## Ключевые особенности
- Удобный поиск и динамически изменяемый интерфейс;
- Гибкая админ-панель для редактирования и добавления товара;
- Легкость в использовании
